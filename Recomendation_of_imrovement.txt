Чтобы адаптировать игру под тему "Космическая арена", мы заменим фон и изображение цели. Вот как можно изменить ваш код:

1. **Заменим фон**:
    - Вместо `screen.fill(color)` будем загружать и отображать изображение фона.

2. **Заменим изображение цели**:
    - Вместо `target_duck.png` используем изображение космической тарелки.

Вот обновлённый код:

```python
import pygame
import random

pygame.init()

SCREEN_WIDTH = 800
SCREEN_HEIGHT = 600
screen = pygame.display.set_mode((SCREEN_WIDTH, SCREEN_HEIGHT))

pygame.display.set_caption("Shooting Gallery")
icon = pygame.image.load("img/icon.png")
pygame.display.set_icon(icon)

# Загрузка изображений
background_img = pygame.image.load("img/space_background.png")  # Убедитесь, что путь и имя файла верны
target_img = pygame.image.load("img/target_ufo.png")  # Убедитесь, что путь и имя файла верны

target_width = 210
target_height = 80

target_x = random.randint(0, SCREEN_WIDTH - target_width)
target_y = random.randint(0, SCREEN_HEIGHT - target_height)

running = True
while running:
    # Отображение фонового изображения
    screen.blit(background_img, (0, 0))

    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False
        if event.type == pygame.MOUSEBUTTONDOWN:
            mouse_x, mouse_y = pygame.mouse.get_pos()
            if target_x < mouse_x < target_x + target_width and target_y < mouse_y < target_y + target_height:
                target_x = random.randint(0, SCREEN_WIDTH - target_width)
                target_y = random.randint(0, SCREEN_HEIGHT - target_height)

    # Отображение цели
    screen.blit(target_img, (target_x, target_y))
    pygame.display.update()

pygame.quit()
```

### Что нужно сделать:
- Поместите изображение фона в папку `img` и назовите его `space_background.png`.
- Поместите изображение космической тарелки в папку `img` и назовите его `target_ufo.png`.

Убедитесь, что пути к файлам изображений указаны правильно, чтобы игра могла их загрузить. После этих изменений игра будет иметь космическую тематику.
